# Хентай бот
Следуя инструкции у вас будет собственный телеграм бот в которого вы сможете загружать свои материалы, после чего бот будет рандомно их показывать

## Требования
- У вас должен быть установлен Python версии 3.Х
- Создайте файл "SaA_memory.txt" там где будет находиться бот, или отредактируйте код что бы изменить название

## Инструкция
- Скачайте архив
- Распакуйте куда угодно
  (Файл бота и файл памяти должны находиться в одной директории)
- Установите зависимости:
```pip install -r requirements.txt```
- Откройте файл бота в любом редакторе
- Замените токен от @botFather и канал на свои
- Запустите бота ```python st_saa_bot_hn5.py```
